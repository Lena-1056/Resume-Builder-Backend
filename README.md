# Resume-Builder-Backend
Spring Boot backend for an AI Resume Builder that generates ATS-optimized resumes, supports secure JWT authentication, email verification, and seamless API integration.

# AI Resume Builder - Backend

## 🚀 Overview
Backend service for AI Resume Builder with authentication and email verification.

## 🔧 Tech Stack
- Spring Boot
- Spring Security (JWT)
- PostgreSQL
- Java Mail Sender

## 🔐 Features
- User Registration & Login
- JWT Authentication
- Email Verification
- Secure REST APIs

## 📌 API Endpoints
- /api/auth/register
- /api/auth/login
- /api/auth/verify-email

## ⚙️ Setup Instructions
1. Clone repo
2. Configure application.properties
3. Run application
